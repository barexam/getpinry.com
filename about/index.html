The Open Source Pin Board

Pinry was the first and remains the best open source pin board alternative to sites like Wookmark and Pinterest. These sites are great for sharing and interacting with other people but if something goes wrong or the service shuts down you lose everything. Pinry allows you to host your own board knowing that it will always be there.

Contacting The Developers

The best way to get help with Pinry is using our GitHub issue tracker. Alternatively you can email us but we aren't as quick to respond to those.

  • Pinry issue tracker: Issue Tracker
  • Pinry email address: devs [at] getpinry [dot] com

Deploying Pinry

Our supported and suggested way to deploy Pinry is using Docker. We provide support and instructions for that over at the docker-pinry GitHub repository. If you'd like a different setup then check out the hundreds of tutorials for production Django deployment found via Google.

  • docker-pinry code: GitHub
  • docker-pinry issue tracker: Issues

Want to be a contributor?

The best way is to clone our GitHub repo, build out a feature or fix some bugs in a clean style that matches our own and if we like it we'll merge it! If you continuously provide Pinry with good updates we'll contact you or you can contact us with your track record and we can discuss getting you commit privileges.
